## 1. Elo 레이팅

### 1.1 기대 승점율 (expected score)

홈팀의 경기 전 기대 승점율:

$$E_{\text{home}} = \frac{1}{1 + 10^{-(R_{\text{home}} + H - R_{\text{away}})/400}}$$

```
E_home = 1 / (1 + 10^(-(R_home + H - R_away)/400))
```

- $R_{\text{home}}, R_{\text{away}}$ : 경기 전 두 팀의 Elo 레이팅 (초기값 1500)
- $H$ : 홈 어드밴티지. 중립 경기면 $H=0$, 아니면 $H=100$
- 분모의 **400** : "레이팅 400점 차이 = 승률 약 10배"가 되도록 정한 스케일 상수.
  Elo 체계의 정의이며 튜닝 대상이 아니다.
- $E$는 승=1, 무=0.5, 패=0을 섞은 **기대 승점율**이지 승리 확률이 아니다.
  (무승부 확률 분해는 §2에서 별도로 함)

원정팀 기대값은 $E_{\text{away}} = 1 - E_{\text{home}}$.

### 1.2 레이팅 갱신 (update)

$$R_{\text{home}}^{\text{new}} = R_{\text{home}} + K \cdot G \cdot (S - E_{\text{home}})$$
$$R_{\text{away}}^{\text{new}} = R_{\text{away}} - K \cdot G \cdot (S - E_{\text{home}})$$

```
ΔR = K · G · (S − E_home)
R_home += ΔR ;  R_away −= ΔR     (제로섬)
```

- $S$ : 홈팀의 실제 결과. 승 $1$, 무 $0.5$, 패 $0$.
- 핵심은 $(S - E_{\text{home}})$ — **놀라움(surprise)** 항이다. 예상대로면 0에 가까워
  레이팅이 거의 안 움직이고, 이변일수록 크게 움직인다. 베이지안 갱신의 단순한 형태.
- 갱신은 제로섬: 홈이 얻은 만큼 원정이 잃는다.

### 1.3 경기 중요도 가중치 $K$

| 대회 종류 | $K$ |
|---|---|
| FIFA World Cup (본선) | 60 |
| 대륙선수권 (Euro, Copa América, Asian Cup, …) | 50 |
| World Cup / 대륙선수권 예선, Confederations Cup | 40 |
| Nations League | 30 |
| 기타 군소 경쟁 경기 | 30 |
| 친선전 (Friendly) | 20 |

정보 가치가 높은 경기일수록 $K$를 키워 레이팅을 더 크게 갱신한다.
친선전은 로테이션·실험 라인업이 많아 $K=20$으로 억제.

### 1.4 골 차 배수 $G$

득점차 $d = |\,\text{home\_score} - \text{away\_score}\,|$ 에 대해:

$$
G(d) =
\begin{cases}
1.0 & d \le 1 \\
1.5 & d = 2 \\
\dfrac{11 + d}{8} & d \ge 3
\end{cases}
$$

```
d ≤ 1 : 1.0
d = 2 : 1.5
d ≥ 3 : (11 + d)/8     # 3골차 1.75, 4골차 1.875, ...
```

"대승은 더 강한 전력 신호"라는 가정의 수치화. eloratings.net 관행을 따름.

## 2. 확률 변환 (Elo 차이 → 승/무/패)

Elo의 $E$는 무승부를 0.5승으로 뭉갠 값이라, 시장과 비교하려면
$P(H), P(D), P(A)$ 세 확률로 분해해야 한다. **다항 로지스틱 회귀**를 쓴다.

### 2.1 입력 피처

단 하나:

$$x = \Delta_{\text{pre}} = (R_{\text{home}} + H) - R_{\text{away}}$$

반드시 **경기 전** 레이팅으로 계산한다(데이터 누수 방지). 코드의 `elo_diff_pre`.

### 2.2 모델

클래스 $c \in \{H, D, A\}$ 에 대해 (softmax / multinomial logit):

$$P(c \mid x) = \frac{\exp(\beta_c\, x + \alpha_c)}{\displaystyle\sum_{c' \in \{H,D,A\}} \exp(\beta_{c'}\, x + \alpha_{c'})}$$

```
P(c | x) = softmax_c( β_c · x + α_c )
```

- 계수 $\beta_c, \alpha_c$ 는 과거 경쟁 경기(친선 제외, 1990년 이후)로 적합.
- scikit-learn `LogisticRegression`(multinomial)으로 학습, `data/prob_model.pkl`에 저장.

---

### 2.3 후보 모델들 (멀티모델 비교)

운영 모델은 §2.2 로지스틱이지만, `models.py` 에 여러 후보를 등록해
동일 walk-forward로 공정 비교한다(`compare_models.py`). 채택은 Brier
개선으로만 — 등록 ≠ 채택.

- **M1 Elo-로지스틱** (기준): §2.2 그대로. 피처 `x = Δpre`.
- **M2 Elo-로지스틱 +|diff|**: 피처 2개 `[x, |x|]`. 접전(차이≈0)에서 무승부가
  늘어나는 비선형성을 명시 피처로 포착 시도.
- **M3 Davidson 무승부 모델**: 학습 파라미터 1개(ν)뿐인 해석적 모델.

$$g = 10^{\,x/400}, \qquad
P(H)=\frac{g}{g+1+\nu\sqrt{g}},\;
P(D)=\frac{\nu\sqrt{g}}{g+1+\nu\sqrt{g}},\;
P(A)=\frac{1}{g+1+\nu\sqrt{g}}$$

```
g = 10^(x/400)
분모 D = g + 1 + ν·√g
P(H)=g/D,  P(D)=ν·√g/D,  P(A)=1/D
```

ν 는 학습기간 로그우도를 최대화하도록 1차원 탐색으로 적합. ν가 클수록
무승부 비중↑. 무승부를 별도 자유파라미터로 직접 모델링한다는 점이 로지스틱과
다르다.

검증 결과(같은 walk-forward, 낮을수록 정확): M2 0.5053 · M1 0.5056 ·
M3 0.5065. 세 모델 차이가 미미하다는 것 자체가 "Elo 차이 한 피처에 정보
대부분이 압축돼 있다"는 증거다.

## 3. 평가 지표 — 멀티클래스 Brier Score

예측 확률 벡터 $\mathbf{p} = (p_H, p_D, p_A)$ 와 실제 결과 원-핫
$\mathbf{y} \in \{(1,0,0),(0,1,0),(0,0,1)\}$ 에 대해, $N$경기 평균:

$$\text{Brier} = \frac{1}{N} \sum_{i=1}^{N} \sum_{c \in \{H,D,A\}} \left(p_{i,c} - y_{i,c}\right)^2$$

```
Brier = mean_i Σ_c (p_{i,c} − y_{i,c})²
```

- 완벽한 예측 = $0$. 항상 $(\tfrac13,\tfrac13,\tfrac13)$로 찍으면 $= \tfrac{2}{3} \approx 0.667$.
- 적중 여부가 아니라 "확률을 정직하게 말했는가"를 측정 → 베팅 맥락의 핵심 지표.

**검증 결과 (walk-forward, 학습 1990–2023 / 검증 2024–2026.6):**

| 방식 | Brier |
|---|---|
| uniform (항상 ⅓) | 0.6667 |
| base-rate (학습기간 H/D/A 비율) | 0.6385 |
| **Elo + 로지스틱 (본 모델)** | **0.5056** |

## 4. 시장 내재 확률 (마진 제거)

### 4.1 소수 배당(decimal odds)에서

마감 배당 $o_H, o_D, o_A$ (예: 1.85) 의 역수를 합으로 정규화해 북메이커 마진 제거:

$$p_c = \frac{1/o_c}{\dfrac{1}{o_H} + \dfrac{1}{o_D} + \dfrac{1}{o_A}}$$

```
p_c = (1/o_c) / Σ_k (1/o_k)
```

이 $\mathbf{p}$ 와 모델 $\mathbf{p}$ 를 **같은 경기들**에 대해 §3 Brier로 직접 대결한다.
모델 Brier < 시장 Brier 이면 "시장보다 정확"의 실증.

### 4.2 미국식 배당(우승 마켓)에서

우승 확률 마켓의 +배당 환산 내재 확률:

$$p = \frac{100}{\text{odds} + 100}$$

(예: +470 → $100/570 \approx 17.5\%$. 여러 팀 합산이 100%를 넘는 것이 북메이커 마진.)

## 5. 토너먼트 진출 확률 (몬테카를로)

조별리그 72경기 확률로 결과를 추첨하고, 전체 대회를 $N = 20{,}000$회 시뮬레이션.

### 5.1 한 경기 추첨

균등난수 $u \sim U(0,1)$ 로:

$$
\text{결과} =
\begin{cases}
\text{홈승} & u < p_H \\
\text{무} & p_H \le u < p_H + p_D \\
\text{원정승} & u \ge p_H + p_D
\end{cases}
$$

승점 부여: 승 3, 무 1·1, 패 0. (rng seed = 42, 재현성)

### 5.2 토너먼트(녹아웃) 승자 진출 확률

스코어 모델이 없으므로 Elo 기대승점율을 진출 확률로 근사 (연장·승부차기 개념 포함):

$$P(a \text{ beats } b) = \frac{1}{1 + 10^{-(R_a - R_b)/400}}$$

### 5.3 우승 확률 추정

$$\hat{P}(\text{champion} = t) = \frac{(\text{$t$가 우승한 시행 수})}{N}$$

표준오차(이항): $\text{SE} = \sqrt{\hat p(1-\hat p)/N}$.
$\hat p = 0.16,\ N = 20{,}000$ 이면 $\text{SE} \approx 0.0026$ ( $\pm 0.26\%p$ ).

### 5.4 명시적 근사 두 가지

- **(a) 타이브레이크**: 조 동률 시 골득실 대신 Elo로 정렬. (모델이 승무패만 출력)
- **(b) 32강 대진**: 공식 브래킷 대신 "조 1위 풀 vs 2·3위 풀" 무작위 추첨.
  공식 브래킷의 경로 난이도 효과가 사라져 중상위권에 ±1%p 노이즈.
